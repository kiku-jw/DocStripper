Quick Start

1

Clone the Repository

git clone https://github.com/kiku-jw/DocStripper2.git
cd DocStripper2
2

Make Executable (Optional)

chmod +x tool.py
3

Start Cleaning

python tool.py document.txt

Usage Examples

Clean a Single File

python tool.py document.txt

Preview Changes

python tool.py --dry-run document.txt

Batch Processing

python tool.py *.txt *.docx

Undo Changes

python tool.py --undo
